Building a GitHub Page with theme using Hugo (Windows)

This is my first personal blog built with Hugo, and it took me a really long time to read the official guideline and work through every single details.

Preparation

Since my computer’s system is Windows 10, which makes it less convenient to finish this job, I need to do the following preparations before all steps.

  1. Install Chocolatey to install Hugo

  2. Install Hugo with Chocolatey

  3. Install Typora (Markdown editor)

In this process, a key thing to remember is that whenever cmd/Powershell/Git Bash is used, always “run as the administrator”, especially for Windows system.

Steps

Step 1: Initialize the hugo site

Open cmd and enter the following command under the folder where you want to store your files in the blog.

hugo new site Jishen-blog

Step 2: Choose a theme and modify config.toml file

Open GitHub Page theme page and choose a theme you want, I choose the “ink” theme because I am a fan of simplicity. Enter its GitHub website and copy the website.

cd Jishen-blog
mkdir -p themes
cd themes
git clone https://github.com/xxxx ink #the url you just copied + name

Open config.toml with Visual Studio Code, add the following line

theme = "ink" # whatever name you choose

Step 3: Create an article and preview the website

cd Jishen-blog
hugo new post/my-first-post.md

Edit the md file you created and save. And then, enter the following code in cmd

hugo server -D

By doing this, you can preview your website on http://localhost:1313. If you are not satisfied, you can go back and modify the config file until you’re okay with it. Finally, enter hugo in cmd to create the website folder, stored in “public”.

Step 4: Upload the local file on gitHub

First, go to GitHub and create a new repository with name (user.name).github.io, and then push the local files in public folder.

cd Jishen-blog/public
git init
git remote add origin git@github.com:YinJishen/YinJishen.github.io.git
git status
git add .
git commit -m "Add a new post"
git push -u origin master

Now you’re done! If you still have any question, you are welcome to email me by jishen.yin@duke.edu
