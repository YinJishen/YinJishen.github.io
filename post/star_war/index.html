Oldest Character in Star Wars Series

Source Code: https://github.com/YinJishen/Biostat823-Homework/tree/master/hw5

API documentation: https://swapi.dev/documentation

Download the data

This problem requires us to find all people and all films using the API. Since the url is formatted such as “https://swapi.dev/api/people/32" or “https://swapi.dev/api/films/3".

So in order to get all people and films, I looped from 1 to 100 for “people” and 1 to 10 for “film”. If the url does not work, it will return {“details”: None}. So I can filter this result by checking if “details” is in the json file.

Finally, I got 82 valid people data, and 6 valid films data.

Find the oldest character

The birth year in star war is defined as “XXX BBY” or “XXX ABY”. BBY indicates “before the battle of Yavin”, ABY indicates “after the battle of Yavin”. After filtering all character with unknown age, I found that all people have “BBY” in their birth of year. So I find the oldest people easily – Yoda.

List all films Yoda appeared

This part is easy. Extract the films list of “Yoda”, and left join with “Films” table. The result is:

The Empire Strikes Back

Return of the Jedi

The Phantom Menace

Attack of the Clones

Revenge of the Sith
