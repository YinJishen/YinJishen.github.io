COVID-19 Team Project

This is the final project of BIOSTAT823: Statistical Programming for Big Data in 2020 Fall Semester at Duke. Our team name is “Durham Sloths”. This project is finished by Zhuoran Hou(https://github.com/ZhuoranHou), Qianyin Lu (https://github.com/QianyinLu), Jishen Yin (https://github.com/YinJishen) and Zewen Zhang (https://github.com/zewenzhang1007).

Project GitHub Page: https://github.com/YinJishen/BIOSTAT823-Final-Project

In this project, we mainly work on the analysis on different aspects of COVID-19 data. In part 1, we recreated the overview of COVID-19 in U.S., referencing 1points3acres with the CDC data, updated with real time. In part 2, we used some simple metrics to compare the effect of policies between state, such as face mask mandatory. We also compared the increase rate in each state with the increase rate in the whole U.S. and made the map plot to show it. In part 3, we built a random forest model to predict the increased rate based on hospitalization data, policy data and demographic data. In part 4, we analyzed the individual level data, comparing COVID-19 metric between different race, age and sex. We also included sampling method in part 4, since the whole dataset is quite large.

We built the structure of this project together. So I will simply discuss about the parts that I took responsible for. For data collection and data preprocessing, I worked on the data about hospitalization. It’ not hard, the data is quite clean. What I did was to delete the variables about confidence interval, cause it has high correlation with the estimate variable. It also took me some time to find the codebook of the dataset since it’s hidden behind the data source website.

For the dashboard, each page represents one part in the project. I firstly finished the random forest model and gave satisfactory results, the dashboard part is finished by Presnie Lu. In part 2, I worked on both the coding and dashboard part. When creating the dashboard, it’s hard to create a map plot since it’s my first time to use the map plot with streamlit. Fortunately, Zhuoran worked on part 1, which also contained a map plot. With the help of Zhuoran, I successfully created an interactive map plot.

I think the main challenge of this project is the time. All of us had a busy schedule in these weeks. But still, we can squeeze our time to have meetings and work on this project. I think our project is terrific, even though we could do it much better if we have more time. I sincerely want to thank my teammate for being so efficient and full of team spirit. It’s a great experience for us.
