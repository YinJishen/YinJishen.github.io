Dashboard for Data of PhDs Awarded in the US

Data Source: https://ncses.nsf.gov/pubs/nsf19301/data

GitHub Page: https://github.com/YinJishen/Biostat823-Homework/tree/master/hw6

Dashboard Url: https://demo-case.herokuapp.com/

In this homework, I used two table: table 4 and table 6. The preprocess steps are in preprocess.ipynb file. I mainly used melt and concat function in pandas to change the data from the wide format into long format. For data in table 6, there are some “D” numbers that indicate confidential data, I cleared this data before doing the visualization part in case that the error message arises. After cleaning the dataset into three csv files, I made some interactive bar plots in the dashboard with streamlit.
